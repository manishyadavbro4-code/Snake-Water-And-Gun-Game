# 🐍💧🔫 Snake Water Gun Game

This is a simple Python game based on the classic **Snake, Water, Gun** concept.  
It is similar to "Rock, Paper, Scissors" where:  
- Snake drinks Water 🐍 > 💧  
- Water douses Gun 💧 > 🔫  
- Gun shoots Snake 🔫 > 🐍
